# 📝 Pixel Notes

A stylish desktop notes application built with Electron, featuring a minimalist pixel art aesthetic.

---

## 🕹️ Interface Guide (English)

Since the application uses graphical pixel spheres instead of text labels, the function of each button is described below. The buttons are numbered from **left to right** on the control panel:

| # | Sphere Color | Function Name | What it does |
| :---: | :--- | :--- | :--- |
| **1** | 🔴 Pink | **Create File** | Creates a new blank note |
| **2** | 🟢 Green | **Save** | Saves current changes in the note |
| **3** | 🟣 Purple | **Delete** | Permanently deletes the current (selected) note |
| **4** | 🟠 Orange | **Notes List** | Opens a list of all previously saved notes |

### 💻 Window Controls
Located in the top right corner of the application:
* 🗕 **Minimize** — minimizes the application to the taskbar.
* ❌ **Close** — completely closes the application.

---

## 🛠️ Built With

- **Electron** — for the desktop environment
- **JavaScript (ES6+)** — application logic
- **HTML5 & CSS3** — pixel interface and styling

# 📝 Pixel Notes (На русском)

Стильное десктопное приложение для заметок, созданное на Electron. Интерфейс выполнен в минималистичном стиле с использованием пиксельной эстетики.

---

## 🕹️ Руководство по интерфейсу (Управление)

Поскольку в приложении используются графические пиксельные сферы без текстовых подписей, ниже приведено описание функций каждой кнопки. Кнопки пронумерованы **слева направо** на панели управления:

| № | Цвет сферы | Название функции | Что делает |
| :---: | :--- | :--- | :--- |
| **1** | 🔴 Розовый | **Создать файл** | Создает новую чистую заметку |
| **2** | 🟢 Зеленый | **Сохранить** | Сохраняет текущие изменения в заметке |
| **3** | 🟣 Фиолетовый | **Удалить** | Безвозвратно удаляет текущую (выбранную) заметку |
| **4** | 🟠 Оранжевый | **Список заметок** | Открывает список всех ранее сохраненных заметок |

### 💻 Управление окном
В правом верхнем углу приложения расположены кнопки управления окном:
* 🗕 **Свернуть** — сворачивает приложение в панель задач.
* ❌ **Закрыть** — полностью закрывает приложение.
